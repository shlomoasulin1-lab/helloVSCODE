# פרויקט HelloVSCode

בפרויקט זה יש שני קבצי פייתון:

1. `greet.py`
   - קובץ פשוט שמדפיס למסך את ההודעה:
     `שלום לך שלמה`
   - ניתן להריץ אותו ב־Python כדי לקבל ברכה בפורמט טקסט.

2. `calculator.py`
   - קובץ מחשבון אינטראקטיבי.
   - בזמן ריצה הוא מבקש מהמשתמש:
     - מספר ראשון
     - מספר שני
     - פעולה (חיבור, חיסור, כפל או חילוק)
   - אחרי החישוב הוא מציג את התוצאה.
   - המחשבון רץ בלולאה עד שהמשתמש לוחץ `רווח` ואז `Enter`.

## איך להריץ

ב־PowerShell או בטרמינל בתיקיית הפרויקט:

```powershell
python greet.py
python calculator.py
```

## הערות

- הקבצים במטרה ללמוד כיצד להריץ סקריפטים פשוטים ב־Python.
- ניתן להוסיף קבצים נוספים ולהרחיב את המחשבון בעתיד.
